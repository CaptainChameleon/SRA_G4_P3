
## Commands
- Push code: ```start_robot.ps1 -p```
- Run existing code on the robot: ```start_robot.ps1 -r```
- Get logs: ```start_robot.ps1 -l```
- Run existing code on the robot & Get logs: ```start_robot.ps1 -r -l```-
